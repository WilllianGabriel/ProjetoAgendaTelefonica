# Agenda Telefônica - Text Files Version (Original)

Esta é a versão inicial do projeto **Agenda Telefônica**, feita em Java puro sem banco de dados, utilizando arquivos de texto para armazenar os contatos.

## Funcionalidades
- Cadastrar contatos com nome e número de telefone
- Listar contatos cadastrados
- Editar contatos
- Remover contatos
- Armazenamento dos contatos em arquivo de texto local

## Como executar
1. Abra o projeto em uma IDE Java (Eclipse ou IntelliJ)
2. Execute a classe `Main.java`
3. Siga as instruções no console para interagir com a agenda

## Observações
- Esta versão **não possui interface gráfica**
- Serve como base para evoluções futuras do projeto
